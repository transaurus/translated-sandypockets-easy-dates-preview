# `dateNowISO()`

返回当前日期的 [ISO 8601](https://en.wikipedia.org/wiki/ISO_8601) 标准格式字符串。

:::tip
ISO 是国际通用的日期和时间表示标准。
:::

## 用法

```javascript
import { dateNowISO } from 'easy-dates';

dateNowISO();
```

## 属性类型

该函数无需传入任何属性参数。

## 示例

```javascript
dateNowISO() // "2022-04-02T15:43:01.539Z"
```
