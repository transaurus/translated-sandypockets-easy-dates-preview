# `getTodayName()`

返回当天星期名称。

## 用法

```javascript
import { getTodayName } from 'easy-dates';

getTodayName() // Friday
```

## 属性类型

该函数无需传入任何属性。

| prop | type | required | default |
|------|------|----------|---------|
| n/a  | n/a  | n/a      | n/a     |
