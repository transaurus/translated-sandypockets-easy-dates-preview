# `getYear()`

返回当前年份。该函数不接受任何参数。

## 用法

```javascript
import { getYear } from 'easy-dates';

getYear()
```

## 属性类型

此函数无需传入参数。

| props | type | required | default |
|-------|------|----------|---------|
| n/a   | n/a  |  n/a     | n/a     |

## 示例

```javascript
getYear() // 2022
```
